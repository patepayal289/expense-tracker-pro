# 🚀 KhataPro: Advanced Expense & Ledger Tracker

**KhataPro** is a beautifully designed, premium-grade business ledger and expense tracking web application. It acts as a digital ledger to effortlessly measure precise Cash IN/OUT events, comprehensively monitor individual customer credit history, send automated WhatsApp reminders to clients, and generate robust PDF accounting reports natively.

Designed comprehensively across all viewports to provide a **True Native App Experience**, featuring fluid Micro-Animations, Context-level State Management, intuitive Floating Actions, and Native Mobile interactions.

---

## 🌟 Key Features

### 💰 **Dynamic General Cashbook**
- Seamlessly record daily internal store collections independent from user-specific credits.
- Instant overview of **"Cash In Hand"** vs **"Today's Balance"**.
- Automatically syncs any credit-sale or received payments from any individual *Customer Ledger* directly into the primary *Cashbook* log, creating a perfect dual-entry accounting workflow.

### 👥 **Robust Customer Directory & Ledgers**
- Efficiently track individual debit/credits across hundreds of uniquely organized customer accounts.
- Actionable Dropdowns with instant **WhatsApp Integration** dynamically reminding customers of overdue balances, natively bridging app-functions with third-party messaging.
- Deep ledger timeline layout designed for immediate chronological validation.

### 📊 **Enterprise-Grade Analysis Dashboard**
- Intelligent visual metrics engineered with `react-chartjs-2` summarizing Top Balances into visual Bar Graphs, and analyzing real-time Cashflow ratios.
- Granular Activity Feeds logging every ecosystem action.
- Single-click comprehensive **PDF Report Generation** detailing full business operations securely processed via `jspdf`.

### 📱 **PWA-Ready Mobile Responsive Design**
- Unprecedented responsive UI architecture adapting the desktop interface directly into an **App-like Mobile Interface** using deep layout swaps.
- Seamless conversion of the traditional desktop Sidebar into a floating sticky Bottom Tab-Bar for native device experiences.
- Universal **Dark Mode** & **Light Mode** styling governed seamlessly by deeply injected CSS Variables, completely transforming the visual palette on the fly.

---

## 🛠️ Technology Stack

- **Framework**: `React 19` 
- **Navigation**: `react-router-dom` creating a multi-layer seamless SPA experience
- **State Architecture**: `React Context API` eliminating rigid prop-drilling
- **Data Persistence**: Synchronous continuous `localStorage` Database implementation
- **Styling**: Modernized Custom CSS-Variables layered directly over `Bootstrap 5` utility classes (Glassmorphism, Layered Soft-Shadows, Animated Viewports)
- **Reporting**: Automated `jspdf` & `jspdf-autotable` metric parsing
- **Graphs**: `Chart.js` via `react-chartjs-2`
- **Iconography**: Clean, consistent `lucide-react` SVG integration
- **Announcements**: Interactive beautiful component rendering via `react-hot-toast`

## 🚀 Getting Started

To launch this application on your local machine, run the following commands:

```bash
# Clone the repository
git clone https://github.com/YOUR_GITHUB_USERNAME/khatapro-clone.git

# Navigate into the project directory
cd khatapro-clone

# Install all required React dependencies
npm install

# Instantly boot up the local development server (typically on http://localhost:3000)
npm start
```
live link: https://expense-tracker-pro-lhlv.vercel.app/dashboard
